# Portfolio Lê Thị Hải Yến

Website portfolio cá nhân hóa cho Lê Thị Hải Yến, MSSV 25080214, lớp MAC 6B, Trường Quản trị và Kinh doanh, ngành Marketing và Truyền thông.

## Cấu trúc

- `index.html`: trang chủ dạng campaign cover, profile snapshot, case index và reflection.
- `pages/`: trang chi tiết 6 bài tập và trang tổng kết.
- `files/`: 6 PDF bài tập và `infographic-bai5.pdf`.
- `images/`: favicon và visual campaign board.
- `css/style.css`: giao diện marketing/editorial responsive.
- `js/script.js`: menu mobile, reveal nhẹ và trạng thái điều hướng.

## Chạy local

Mở trực tiếp `index.html` bằng trình duyệt. Không cần backend, framework hoặc build step.

## Ghi chú Git

Repo này được khởi tạo mới độc lập, không dùng lịch sử Git của repo portfolio trước.
